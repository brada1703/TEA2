{{ 'GRAMMAR.FromAdjectives' | translate }}

Adverbs can be formed from adjectives in several different ways. Usually, we simply add -ly to the end of the adjective to form an adverb. However, if the adjective ends in -y, then we should remove the -y and add -ily. If the adjective ends in -e, we should drop the -e and add -ly. If the adjective ends in -ful, we double the l at the end of the word before adding -ly. Finally, if the adjective ends in -ic, we should add -ally.

Examples:

Adjective Adverb
active actively
massive massively
careful carefully
cheerful cheerfully
clear clearly
different differently
healthy healthily
impressive impressively
insulting insultingly
monotonous monotonously
quick quickly
bad badly
lazy lazily
basic basically
economic economically
