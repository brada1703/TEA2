{{ 'GRAMMAR.Order' | translate }}

When using multiple adjectives to describe the same noun, we must put them in a certain order.

Quantity, Opinion, Size, Temperature, Age, Shape, Color, Origin, Material

Type of Adjective Example
Quantity five, seventeen, many, few
Opinion ugly, beautiful, delicious
Size small, big, gigantic
Temperature warm, cool, hot, cold
Age new, young, 21-year-old, old
Shape round, rectangular, square
Color red, orange, yellow, green, blue, indigo, violet
Origin Canadian, Australian, British
Material Carbon fiber, steel, wood

Examples:

  • I have five delicious red tomatoes growing in my garden.
  • She is making some really beautiful clay pottery.
  • He likes to buy bottles of aged brown Kentucky bourbon.
  • Someone drove by in a beautiful new red Italian sports car.
